# Flags
The CPU has several flags that indicate the outcome of certain operations or conditions.

These flags are bits in a special register known as the `flags register` or `EFLAGS register`.

```
CF  -   Carry Flag
OF  -   Overflow Flag
SF  -   Sign Flag
ZF  -   Zero Flag
AC  -   Auxiliary Carry
PF  -   Parity Flag
TF  -   Trap Flag
DF  -   Direction Flag
IF  -   Interrupt enable
```
